# Text-originality-analyzer
This project is a web-based system for detecting plagiarism and AI-generated content.
The current version implements sentence-level plagiarism detection using TF-IDF
similarity and heuristic-based AI writing analysis.

## Features
- Sentence-level plagiarism detection
- AI-generated text analysis
- Explainable feedback on originality
- Flask-based web interface

## Status
Initial baseline implementation. Further enhancements planned.
